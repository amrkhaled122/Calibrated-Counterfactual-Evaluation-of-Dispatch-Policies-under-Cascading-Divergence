The raw and processed Meituan data files are licensed/local artifacts and are not committed by default.

Expected local paths:

- `data/all_waybill_info_meituan_0322_edited.csv`
- `data/dispatch_cycles_with_scaled_recipient_coords.cleaned.csv`
- `data/actual_eta_by_order.csv`
- `data/courier_working_blocks.csv`
- `data/travel_time_model.json`
- `data/features/offers_observations.parquet`
- `data/features/manifest.json`

Place licensed copies at these paths, or use Git LFS / a release archive if distribution is permitted.
